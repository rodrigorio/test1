Publicaciones en Comunidad

El sistema permite crear publicaciones con contenidos relacionados a la discapacidad, información para profesionales y personas relacionas con NEE.
De esta manera se crearan canales de comunicación informativos para la comunidad

Instituciones

Las instituciones son unidades que agrupan integrantes. Las instituciones que son cargadas por los integrantes de la comunidad serán visibles para cualquier visitante.
Un integrante registrado podrá crear una nueva institución en el caso de que no exista. Para poder registrar una institución el integrante debe completar
un formulario con datos correspondientes a una entidad.

Catalogo de Descargas

El sistema contará con un repositorio de software libre organizado por categoría de acuerdo a una problemática o tema específico.
Los integrantes o visitantes podrán descargar el software para su uso libre distribución
